About WeeQuiz

WeeQuiz is a quiz website dedicated to the beautiful country of Scotland. Started in March 2020, it has always had a single aim: to provide free quizzes that allow everyone to think about, explore and get to know Scotland. Think we're missing a trick? Then don't hesitate to Email Us!

The Technology

  • Python: the main language the code is written in
  • Flask: the python web framework I chose
  • Javascript: the fancy stuff
  • Github: where the code is hosted
  • VSCode: where the code it written
  • Bootstrap: making the code pretty, standardised and responsive
  • Azure: what runs the show

The Team

WeeQuiz is a small team at present but with big ambitions to grow in the future.
